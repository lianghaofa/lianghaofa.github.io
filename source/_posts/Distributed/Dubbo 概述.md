## Dubbo 概述

Java RPC框架，三大核心功能：面向接口的远程方法调用，智能容错和负载均衡，服务自动注册和发现。 

基于spring容器运行。

服务与服务之间交换数据。

- mysql,共享表

- 直连，xml - 文件大，性能低，结构严谨，XSD。 

- JSON -> 小，性能高，结构不严谨，可读性差。
  
- 序列化 -> 文件更小，性能最高，面向对象。可读性好。




RPC 远程服务调用
